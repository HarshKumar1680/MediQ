# MediQ 

Medical Treatment Information Retrieval System based on Disease Queries.

## Features
- Inverted Index
- TF-IDF Ranking
- BM25 Ranking
- Ranked Retrieval
- Evaluation Metrics

## Tech Stack
- Python
- Flask
- NLTK
- Rank-BM25

## How to Run

```bash
pip install -r requirements.txt
python run.py
